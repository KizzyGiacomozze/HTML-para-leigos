O que diabos é HTML?

(Voltar)

HTML é uma linguagem de marcação utilizada na web para constuir páginas, é responsável pela estrutura da página, atuando como o esqueleto do conteúdo exibido nos sites.

Como é construído

O HTML é composto por tags que podem conter ou não atributos.

A estrutura base de um HTML é composta por: Head e Body. O Head é responsável por conter as informações que serão carregadas antes do carregamento da página para o usuário, já o Body é composto por tudo que será exibido na página.

Tags de texto

Responsáveis por estruturar os textos da página e estabelecerem uma hierarquia.

Tags tipo h

São as tags utilizadas para títulos, h simboliza header, que é título em inglês. Ela é classificada por números, então temos do h1 até o h6, sendo o h1 o maior e o h6 o menor.

Tag p

A tag p é utilizada para estrutura de parágrafo, sempre que for criar um parágrafo, ou seja, sempre que seu conteúdo tiver uma quebra de linha, você deve inserir um novo p.

Tag bloquequote

É utilizada para escrever citações, criando um recuo no texto.

Tag mark

É utilizada para grifar o texto, atuando como um marca-texto.

Tag sup

Deixa o texto mais acima dos outros textos, como se ele estivesse suspenso.

Tag u

Faz com que o texto fique sublinhado.

Atributos de link

target="_blank"

Atributo utilizado para fazer com que o link abra em outra aba do navegador.

target="_self"

Atributo utilizado para fazer com que o link abra na mesma aba do navegador.

title="exemplo"

Atributo que cria um tooltip no link quando o mouse repousa sobre o link.

Atributos das tags

(Voltar)

São estruturas para serem utilizadas em conjunto com as tags, com o objetivo de auxiliar e facilitar a construção dos sites. Podem ser utilizadas para gerar encontrabilidade quando trabalhamos em conjunto com o JavaScript e o CSS.

Alguns exemplos desses atributos são:

ID

Atributo que funciona como um identificador para a tag.

Style

Atributo que simula uma parte do css e permite mudanças de estilo simples, como mudar a cor de um texto.

Class

Atributo que forma uma classe de elementos, quando editado, altera todos os componentes que estejam dentro da mesma classe.

src

Atributo que auxilia na encontrabilidade de uma imagem.

Width

Atributo que altera a largura de uma imagem ou componente.
